### OMG Mute Me

I don't actually mind seeing ads on Youtube or whatever. Like visually, it's not a big deal for me.

However, the problem for me is the audio is always really mixed badly or obnoxious.

So this is just an extension you can installed yourself to auto-mute ads.

There's probably a dozen ways to make the javascript neater so feel free to steal this and turn it into your own.

## Installation

I really don't want to setup an apple developer account or google developer account and have to worry about keeping up with anything, all that tertiary stuff is so exhausting to me, so here's a list of ways to install the plugin:

1. Safari: https://developer.apple.com/documentation/safariservices/safari_web_extensions
2. Chrome: https://developer.chrome.com/docs/extensions/mv3/getstarted/
3. Firefox: https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/Your_first_WebExtension

If you wanna do all that yourself and publish it, feel free. Just reference me as a contributing author.
